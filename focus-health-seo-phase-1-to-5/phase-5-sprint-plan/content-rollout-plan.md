## Quality Checklist (Per Content Piece)

- [ ] Minimum 1,200 words (new articles: 1,500–2,000)
- [ ] Primary keyword in title, H1, first 100 words, and meta description
- [ ] 4–6 H2 headings with keyword variations
- [ ] 3–5 internal links (at least 1 to pillar page)
- [ ] 1–2 external links to authoritative sources
- [ ] Closing CTA block with link to `/partners` or `/contact`
- [ ] FAQPage schema where applicable
- [ ] Article schema with correct `datePublished` and `dateModified`
- [ ] Featured image with descriptive alt text
- [ ] Meta description between 120–155 characters
- [ ] Title tag between 50–65 characters
- [ ] British English throughout
- [ ] Proofread for grammar, spelling, and factual accuracy
