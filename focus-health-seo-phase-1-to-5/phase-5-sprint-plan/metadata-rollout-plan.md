## Implementation Notes

- Roll out all title tag changes in a single deployment to avoid partial indexing
- After deployment, request re-indexing via Google Search Console for all updated pages
- Monitor CTR in Search Console for 30 days post-deployment to measure impact
- Title tags should be implemented in `metadata` exports in Next.js App Router page files
- Meta descriptions should be set via the `description` field in the same `metadata` export
