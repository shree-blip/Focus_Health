# SEO Blueprint: Metadata Recommendations (All Pages)

---

## Title Tag Standards

| Rule | Requirement |
|------|-------------|
| Length | 50–65 characters |
| Format | `Primary Keyword | Brand Differentiator | Focus Health` |
| Uniqueness | Every page must have a unique title tag |
| Brand position | "Focus Health" at the end |
| Keyword placement | Primary keyword in the first 40 characters |

---

## Meta Description Standards

| Rule | Requirement |
|------|-------------|
| Length | 120–155 characters |
| Format | Action-oriented summary with primary keyword |
| Uniqueness | Every page must have a unique meta description |
| CTA | Include a soft call-to-action where appropriate |

## H1 Tag Standards

| Rule | Requirement |
|------|-------------|
| Count | Exactly one H1 per page |
| Content | Must contain the primary keyword |
| Length | 30–70 characters |
| Uniqueness | Every page must have a unique H1 |
| Relationship to title | H1 should complement (not duplicate) the title tag |

---

## Canonical Tag Standards

| Rule | Requirement |
|------|-------------|
| Presence | Every indexed page must have a self-referencing canonical |
| Format | Absolute URL with `https://www.getfocushealth.com/` prefix |
| Trailing slash | Consistent — no trailing slash on page URLs |
| Status | Currently clean — no issues detected |

---

## Open Graph & Social Standards

| Tag | Requirement |
|-----|-------------|
| `og:title` | Mirror the title tag |
| `og:description` | Mirror the meta description |
| `og:image` | 1200×630px branded image per page (or shared default) |
| `og:url` | Canonical URL |
| `og:type` | `website` for core pages, `article` for insights |
| `twitter:card` | `summary_large_image` |

---

## Notes

- Nine of sixteen indexed pages currently have single-word title tags (e.g., "Platform", "Market", "Leadership") — this is the single highest-impact SEO fix available
- The Investors page currently has the title "Partners" — this is a metadata error and must be corrected
- All recommended titles follow the `Keyword Phrase | Differentiator | Focus Health` pattern
- Character counts include the full title including separators
- Metadata changes should be implemented in a single sprint for consistency (see Phase 5 sprint plan)
